# image
```php
image($width = 640, $height = 480, $type = "business", $text = null)
```
參數說明

`$width` 寬度

`$height` 高度

`$type` 類型

`$text` 文字

### 類型

1. `people` 人物
2. `abstract` 抽象畫
3. `animals` 動物
4. `business` 商業
5. `cats` 貓
6. `city` 城市
7. `food` 食物
8. `nightlife` 夜晚生活
9. `fashion` 時尚
10. `nature` 大自然
11. `sports` 運動
12. `technics` 工藝
13. `transport` 交通
